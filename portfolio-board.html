Ключевые возможности

Транскрипция Whisper

Автоматическая конвертация аудио через ffmpeg и транскрипция через OpenAI Whisper. Поддержка MP3, WAV, OGG, M4A, FLAC.

ИИ-Саммари на GPT-4o

Каждый звонок анализируется GPT-4o: тема, настроение, ключевые моменты, результат звонка, вопрос клиента.

Аналитический дашборд

Общий анализ по сессии: распределение настроений, топ категорий, боли клиентов, рекомендации. Графики на Recharts.

Realtime через SSE

Обработка файлов в реальном времени через Server-Sent Events. Прогресс-бар, текущий шаг, таймер — всё live.

База знаний

Глубокий анализ каждого звонка: оценка оператора, скрипт продаж, цитаты клиентов, автоматический FAQ.

Экспорт PDF и HTML

Готовые отчёты в HTML и PDF для руководства. Кнопка экспорта прямо в интерфейсе аналитики.
